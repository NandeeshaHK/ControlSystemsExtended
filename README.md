# ControlSystemsExtended

Link to Basic Control systems (From *Lolbouy*): https://github.com/lolbuoy/controlsystems

## Installation

You can install this package via pip using the git link:

```bash
pip install git+https://github.com/NandeeshaHK/ControlSystemsExtended.git
```

Or install locally:

```bash
pip install .
```

## Usage

You can import modules individually:

```python
from control_systems_extended import Auto_mission
from control_systems_extended import GPT_controls
```

Or import everything:

```python
from control_systems_extended import *
```

### Running Scripts

To run the scripts directly, you can use:

```bash
python -m control_systems_extended.Auto_mission
python -m control_systems_extended.mavlink_basics
```
